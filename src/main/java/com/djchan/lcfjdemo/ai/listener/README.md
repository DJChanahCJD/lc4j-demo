## AI Service 和 Chat Model 在可观测性的对比

| 维度         | AI Service可观测性                       | Chat Model可观测性                      |
| ------------ | ---------------------------------------- | --------------------------------------- |
| **观测对象** | 整个AI应用系统（微服务、数据库、网络等） | 大语言模型本身的行为和输出              |
| **关注重点** | 系统性能、可用性、成本控制               | 模型行为、输出质量、语义正确性          |
| **核心指标** | 延迟、错误率、吞吐量、资源利用率         | Token消耗、响应质量、幻觉检测、偏见识别 |
| **技术实现** | 传统监控工具（日志、指标、追踪）         | 专门针对LLM的观测工具和框架             |
| **数据粒度** | 系统调用级别                             | Token级别、语义级别                     |
